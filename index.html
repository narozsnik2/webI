Webprogramozás-1 Előadás Házi feladat

  • Főoldal

Ez az alkalmazás a Webprogramozás-1 tantárgy keretében készült házi feladat megvalósítása. A projekt célja a modern webfejlesztési technológiák gyakorlati bemutatása, különös tekintettel a kliensoldali és szerveroldali működésre. Az alkalmazás több különálló modulból áll, amelyek különböző technológiákat és megközelítéseket mutatnak be, például hagyományos JavaScript DOM-kezelést, SPA (Single Page Application) megoldásokat, React alapú felületet, valamint API kommunikációt Fetch és Axios segítségével. A rendszer célja a CRUD műveletek (létrehozás, olvasás, módosítás, törlés) gyakorlati megvalósítása, valamint az adatok dinamikus megjelenítése böngészőben.

Készítette: Narozsnik Nikolett (E3WVTO)
